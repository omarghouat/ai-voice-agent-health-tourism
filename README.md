# AI Voice Agent for Health Tourism

A web-based voice assistant to help international patients explore, book, and manage health tourism services.

## 🧩 Features

- Speech-to-Text (Vosk)
- LLM-powered responses (OpenAI or Ollama)
- Text-to-Speech output
- Streamlit UI
